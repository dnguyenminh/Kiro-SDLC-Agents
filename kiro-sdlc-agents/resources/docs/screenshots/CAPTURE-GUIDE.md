# Screenshot Capture Guide

Save screenshots to this folder with the following names:

## Required Screenshots

| Filename | Page | What to Capture |
|----------|------|-----------------|
| `kb-dashboard.png` | localhost:3200/dashboard | Full page: Health Score gauge, Recommendations, Due Reviews table, Trends charts |
| `kb-graph.png` | localhost:3200/ | Graph tab: 3D knowledge graph with nodes and edges visible |
| `kb-tags.png` | localhost:3200/tags | Tags page: Tag cloud + taxonomy tree |
| `kb-quality.png` | localhost:3200/quality | Quality page: Distribution chart + low-quality table |
| `kb-analytics.png` | localhost:3200/analytics | Analytics page: Search volume trend + popular queries |
| `kb-sessions.png` | localhost:3200/ (Sessions tab) | Sessions tab: Session list with agent names |
| `kb-browser.png` | localhost:3200/ (Browser tab) | Browser tab: Entry list with filters |
| `kb-stream.png` | localhost:3200/ (Stream tab) | Stream tab: Real-time operations |

## Tips

- Use browser full-screen (F11) for cleaner screenshots
- Dark theme is default — looks great in README
- Recommended width: 1200-1400px browser window
- Use PNG format for crisp text
- Crop browser chrome (address bar, tabs) for cleaner look — or keep it to show URL
